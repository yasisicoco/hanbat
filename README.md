## ✅ 사용 방법
1. 본인 이름의 폴더를 생성한다.
2. 생성된 폴더에 월별 폴더를 생성한다.
3. 알고리즘 풀이 후 월별 날짜에 맞게 파일을 업로드한다.

<br/>

## 📁 Repository 폴더 구조
```
{이름}/{월}/{날짜}/{코드}
```

- 💡 예시: `나싸피 / 2월 / 0212 / 코드`

<br/>

## ❗ 코드 제목 양식

```
{플랫폼_[난이도]문제번호_문제제목.언어} - {기타 내용}
```

- 💡 예시: `BOJ_[실4]1219_문제제목.py`
